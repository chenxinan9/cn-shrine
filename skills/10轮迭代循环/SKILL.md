# 10 轮迭代循环(通用 · Iron Loop 抽专业版 · v1.0)

## 核心定位

→ 抽自开发员 Iron Loop v4.0 100 轮范式, **通用化**用于任意持续改善任务(spec / SKILL / 文档 / 知识库 / 协议)
→ 10 轮 = 1 个任务 / 每轮 30-90 min / 总投入 5-15 小时 = 单任务深度训练

## 触发场景

- "[10轮迭代循环]" / "[Iron Loop 通用]" 显式
- 大任务持续改善(spec 升级 v0→v10 / SKILL 完善 / 文档反复打磨)
- 单文件需 ≥ 3 轮迭代到位
- 反复同类错( 极端化变体 / 漏不闭环)→ 必走

## 不适用

- 单次回复型任务(用[统一沟通回复模版] 13 项即可)
- 全新创建无前作(用[信息整理开发] 7 步)
- 紧急修补 < 30 min(直接动手)
