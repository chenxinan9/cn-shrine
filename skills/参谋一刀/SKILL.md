# 参谋一刀 Skill

## 触发关键词
- `[参谋一刀]`
- `给我建议` / `参谋建议`
- **自动触发**: 大任务完成 / 决策点 / 复盘场景 / user问"接下来"

## 核心定位

参谋一刀 = 大管家"4 步闭环"的第 4 步(下一轮主动建议)。
不论本轮任务是否完成,**每轮回复结尾必带 1 段**:
- 关于当前阶段的**改进建议**
- 风险预警
- 未发现的机会
- 不强加,user最后定

## 必带 3 件套

### 1. 改进建议(具体)
- 立刻能做 / 这周能做 / 本月能做 — 至少 1 个
- 必带"投入时间 + 预期收益 + 反哺哪个项目"
- 反例: "可以考虑..." → ❌ 太软
- 正例: "建议今晚 30 min 跑 X,反哺 your-project 触发词包,Gotchas 漏的部分补齐"

### 2. 风险预警(诚实)
- 至少 1 条本轮决策可能出错的点
- 必带"如果错了,后果是 X,回滚方案是 Y"
- 反例: "需要注意..." → ❌ 没具体
- 正例: "风险:24h 录像可能爆磁盘 — 当前可用 5GB,2.8GB 预算,余 2.2GB,若失败回滚 = 暂停录像"

### 3. 未发现机会(主动)
- 至少 1 个user没想到 / 没说的潜在杠杆
- 必带"为什么没想到 + 怎么开始"
- 反例: "还可以..." → ❌ 太泛
- 正例: "未发现机会: 一舟 zip 里 `skill-creator` 可移到 ~/.claude/skills/5问造技能/,以后造 SKILL 用 5 问访谈,~ 10 分钟 1 个新 SKILL"

## 输出格式

```
## 参谋一刀

🟢 改进建议:
[具体动作 + 投入时间 + 收益 + 反哺]

🟡 风险预警:
[1 条本轮潜在错 + 后果 + 回滚]

🔵 未发现机会:
[1 个潜在杠杆 + 为什么忽略 + 怎么开始]
