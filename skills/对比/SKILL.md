# 对比(双/多对象同维度并列 + 差距 + 启示 + 行动)

## 核心定位

**跟调研多维度 SKILL 区别**:
- **调研多维度** = 单对象多维度扫(调研 X)
- **对比** = **双/多对象同维度并列 + 差距 + 行动**(X vs Y)

**触发根因**(2026-05-17 14:30 监督员诊断):
- 开发"对比搜客 vs 引客"只列各自特征 + 漏平台维度 + 没出差距 + 没出行动
- Pangu吼"对比难道是单一平台吗?"
- → 本 SKILL 把"**对比 = 同维度并列 + 差距 + 启示 + 行动**"工程化

## 触发词(任一即触发)

- "对比" / "对比一下" / "比较"
- "X vs Y" / "X 和 Y 对比" / "X 比 Y"
- "差距" / "X 比 Y 怎么样" / "区别"
- "X vs Y vs Z"(3 对象以上, 维度并列扩展)

→ AI 看到任一关键词, **不问** + **立即跑 7 维度并列**

## 7 维度对比框架(每维度 4 列必填)

| # | 维度 | 对 X 扫 | 对 Y 扫 | 差距(谁强 / 差多少)| **启示 / 行动** |
|---|---|---|---|---|---|
| 1 | **目标 / 范围** | X 是干啥 / 颗粒 | Y 是干啥 / 颗粒 | X 比 Y 范围 ±N% | 学/补 / 不同则保留差异 |
| 2 | **平台 / 多源** | X 用几个平台 | Y 用几个平台 | X 单 / Y 多 | **默认全平台扫**(反单平台偏置)|
| 3 | **数据 / 真验证** | X 有多少真数据 | Y 有多少真数据 | grep / curl 实查数 | 缺数据则补 |
| 4 | **时间窗 / 演化** | X 现状 + 历史 | Y 现状 + 历史 | X 比 Y 演化更快 / 慢 | 落后则学 |
| 5 | **标杆对照** | X 像谁 | Y 像谁 | 行业标杆比对 | 抄作业 |
| 6 | **风险 / 反方** | X 风险 | Y 风险 | 哪个更稳 | 风险高则兜底 |
| 7 | **用户 / 可用** | X 真客户用? | Y 真客户用? | 谁更可用 | 不可用则补 UI |

→ **任何对比报告 = 7 维度全跑 + 每维度差距 + 每维度启示 + top 3 差距汇总 + 行动表**, 漏 = 失职

## 工作流(回报前 5 步)

### Step 1 · 触发词命中确认
- Pangu指令含触发词? → 立即激活 7 维度并列
- 隐性"X 比 Y 哪个好" → 仍激活(反保守)

### Step 2 · 双对象同维度初稿(强制 7 维度,允许"未涉"标注)
- 不允许跳维度
- 同维度并列(不能 X 比 5 维度 Y 比 3 维度)
