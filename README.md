## 5 分钟快速上手

```bash
# 克隆
git clone https://github.com/your-account/cn-shrine.git
cd cn-shrine

# 复制 1 个神到你的项目改名用
cp -r 业务区/示例-SaaS-项目/唐僧\(总管\) ~/your-project/your-pm

# 改 SOUL 改 readme · 用 Claude Code 打开
claude
```

每个神有 **9 公文**:readme / soul / user / agent / claude / memory / TOOLS / workspace / skills

## English

**cn-shrine** is a multi-agent governance framework for solo founders and small teams, grounded in Chinese mythology. 21 agents with classical origins (Shanhaijing, Zhuangzi, Shiji, Buddhist sutras) and internalized virtues — not external rules.

3 tiers: Enterprise (profit) → State (compliance, deprecated) → **Celestial** (all-being mission, this repo).

See [docs/三界跃迁.md](docs/三界跃迁.md) and [docs/21神仙典籍溯源.md](docs/21神仙典籍溯源.md).
