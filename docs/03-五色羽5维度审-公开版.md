# 五色羽 5 维度审 · 公开版 SOP

> 凤凰五色羽:视觉(赤) / 听觉(黄) / 文字(青) / 交互(白) / 留白(玄) · 5 维齐审 = 美学完整

## 一、5 维度速查

| 维度 | 内容 | 真物检 |
|---|---|---|
| **视觉(赤)** | 颜色 / 字体 / 图像 / 排版 | WebFetch 真访 + 截图 + DESIGN token grep |
| **听觉(黄)** | 视频音乐 / 配音 / 节奏(若有) | 视频真听 + 节奏自评 5 秒 |
| **文字(青)** | 文案 voice / 用词 / 句式 / 调性 | 真读 + 反 AI 味 grep + 客户视角 |
| **交互(白)** | UI / UX / 客户路径 / 转化漏斗 | WebFetch 流程跑 + 客户视角 |
| **留白(玄)** | 不说什么 / 不做什么 / 信息密度 / 节奏 | 字数 / 元素数 / 反"堆满" |

## 二、5 步流程

```
Step 0 · date + 真物拉(WebFetch 子页 / 截图 / DESIGN grep)
Step 1 · 视觉(赤) — DESIGN token 守? accent 克制? emoji 数?
Step 2 · 听觉(黄) — 若有视频 / 配音 · 否则跳
Step 3 · 文字(青) — grep 反 AI 味词 · voice 跟项目调性一致?
Step 4 · 交互(白) — 客户路径 5 秒跑 · 操作步数算
Step 5 · 留白(玄) — 信息密度算 · 反"堆满"扫
→ 5 维齐审 → 一刀(利 ≥ 1 / 害 ≥ 1 / 守底线 ≥ 1)
```

## 三、5 维评分(0-10 / 维 · 满分 50)

| 维 | 5 分基线 | 7 分良好 | 9 分优秀 |
|---|---|---|---|
| 视觉 | token 守 50% | 守 75% + accent 克制 | 守 95% + 对标无破调 |
| 听觉 | 节奏中性 | 跟 voice 协奏 | 听觉可独立感受品牌 |
| 文字 | 反 AI 味 grep = 0 | + 短句 + 调性一致 | + 真物故事 ≥ 1 |
| 交互 | 操作 ≤ 5 步 | ≤ 3 步 + 留白齐 | 5 秒第一印象 = 品牌 |
| 留白 | 密度 ≤ 70% | ≤ 50% + 段段喘气 | ≤ 40% + 反"堆满"模范 |

## 四、5 类红色破调(必停 + 升级)

1. emoji 过载(单页 > 10 个)
2. 蓝色 CTA(IT 感)
3. 大块金底(土豪)
4. AI 味词(震惊 / 改变一切 / 完美 / 必然)
5. 信息塞满(密度 > 80%)

→ 红 1 项 = 立即停 + 候上游审

## 六、反阿谀 + 反美学高大上(自检)

跑完审前问自己:
- 没"完美" / "必然" / "唯一" / "一定"?
- 没"元认知架构师"等大词?
- 真物 ≥ 1 / 维(WebFetch + 截图 + grep)?
- 5 维度齐审 · 不偷懒跳维?
- 利害底齐 · 不只夸不批?

→ 5 问全过 = 审通过
