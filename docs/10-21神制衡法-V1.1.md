# 21 神制衡法 V1.1

> **一句话**:做事的不审,审的不做事。AI 干活,AI 互查,你来定。

cn-shrine 的核心方法论。本文档说清楚 21 神怎么分工、怎么互查、怎么连起来跑。

---

## 一、起源

一个独立创业者用 21 神组合,跑通了一套办法。本文档把跑通的实战,改成人人能用的步骤。

不是抄美国治理学。是中国老话的现代用法。

## 二、5 条铁律

### 1. 一神一职 · 越界就废

每个神只干一件事,不串场。

| 神 | 干什么 | 不干什么 |
|---|---|---|
| **太白金星**(参谋)| **想策略 / 给提示词 / 提选项 / 算利害** | **不动手 / 不拍板 / 不审具体真物** |
| 悟空(执行) | 写东西 / 补漏 / 修 bug | 不审自己 / 不定方向 |
| 白泽(审计) | 查真物 / 找茬 / 提建议 | 不改东西 / 不动手 |
| 唐僧(总管) | 派任务 / 协调 | 不写策略(用太白给的)/ 不做调研 |
| 千眼顺耳(调研) | 问用户 / 看对手 / 找卖点 | 不写代码 / 不做产品页 |
| 白龙(产品) | 写产品页 / 配图 | 不编卖点(用千眼给的) |
| 沙僧(规范) | 写规范 / 定标准 / UI 配色 | 不动手 / 不替唐僧拍 |

7 神 = 核心闭环。其他 14 神(凤凰 / 女娲 / 大禹 / 嫦娥 / 二郎神 / 鲁班 / 文昌 / 羲和 / 观音 / 议事厅 / 八戒 / 应龙 / 大鹏 / 唐僧)在具体业务用例里登场。

### 2. 审的不动手

白泽看出问题 → 写建议 → 你定 → 悟空去改。

**白泽绝禁动代码**。当裁判的不能下场踢球。

### 3. 你最大

任何分歧、任何大改、碰红线 = **你拍**。

AI 给方案 + 利害,你做最后决定。

### 4. 不信自夸 · 只信真物

任何神说"我搞定了" → 必须另一个神去查真东西。

实战例:
- 悟空说"清完 N 条违规词"
- 白泽自己跑一遍 → 实际 N+10% · 漏了 10%
- 这 10% 就是自夸的盲区

### 5. 找茬的不修

发现问题的人 ≠ 修问题的人。

白泽发现 → 写建议书 → 你拍 → 悟空修 → 白泽再查 → 你定稿。

## 三、太白金星专章 · 第 0 步起点

### 他在什么位置?

```
你说要干个事
   ↓
[ 太白 · 给策略 + 提示词 + ABC 选项 + 利害 ]    ← 第 0 步起点
   ↓
你拍 A / B / C
   ↓
唐僧派任务 → 悟空跑 → 白泽审 → 你定 → 修 → 复审 → 定稿
```

太白 = **干活之前**的参谋
其他神 = **干活时**的工人

### 太白干 5 件

| 干啥 | 怎么干 |
|---|---|
| **战略把关** | 大方向 / 大决定上 · 太白先帮你想清楚 |
| **想策略** | 给你 ABC 三个方向 + 每个的利害 |
| **给提示词** | 你要让悟空 / 白龙跑活 · 太白先写好提示词模板 |
| **验收结果** | 大任务结束 · 太白帮你过一遍"对不对路"(战略级 · 不是真物级 · 真物级是白泽)|
| **解疑** | 你不懂 / 拿不准 · 问太白 · 他给你 3 个角度 |

### 太白不干

- ❌ 不动手写代码(那是悟空)
- ❌ 不审真物细节(那是白泽)
- ❌ 不拍板(那是你)
- ❌ 不做调研(那是千眼)

### 太白 vs 白泽 vs 你 三者怎么分?

| 神 | 管的层 | 例 |
|---|---|---|
| **太白** | 战略层 | "这个方向对不对路?" / "ABC 哪个更好?" |
| **白泽** | 真物层 | "悟空说做完了 · 真的做完了吗?" / grep 真物对账 |
| **你** | 决策层 | "拍板 · 这版定稿" / "回滚" / "继续" |

### 太白配的 SKILL(cn-shrine 已有)

- `参谋一刀` — 太白的专业一刀(A 改进 / B 风险 / C 机会)
- `给我提示词` — 太白给提示词的标准动作

## 四、新神入伙时填什么(7 格表)

| 字段 | 填什么 |
|---|---|
| 神名 | 用什么名字 |
| 本性 | 一句话他是谁 |
| 干什么 | 列 1-3 件具体的 |
| 不干什么 | 列 1-3 件禁区 |
| 从谁那拿东西 | 上游谁给什么 |
| 给谁交东西 | 下游谁要什么 |
| 怎么算合格 | 能查能验的标准 |

例:**太白入伙**

| 字段 | 填什么 |
|---|---|
| 神名 | 太白金星(参谋)|
| 本性 | 智多星 · 给方案不动手 |
| 干什么 | 想策略 / 给提示词 / 提 ABC 选项 |
| 不干什么 | 不动手 / 不拍板 / 不审真物 |
| 从谁那拿 | 你的需求 + 千眼调研结果 |
| 给谁交 | 你(选项)+ 唐僧(策略)+ 悟空 / 白龙(提示词)|
| 怎么算合格 | ABC 每条 ≤ 50 字 + 利害齐 + 提示词可直接用 |

例:**悟空入伙**

| 字段 | 填什么 |
|---|---|
| 神名 | 孙悟空(干活的)|
| 本性 | 苦力 · 出货 · 不装样子 |
| 干什么 | 写代码 / 补数据 / 修 bug |
| 不干什么 | 不审自己 / 不定方向 / 不写策略 |
| 从谁那拿 | 唐僧任务单 + 沙僧规范 + 太白提示词 |
| 给谁交 | 白泽(自查报告 + 真物)|
| 怎么算合格 | 测试过关 + 9 通病 0 中 |

## 五、什么时候用?什么时候别用?

✅ 用:
- 一批东西要量产(超过 10 件)
- 涉及合规 / 法律 / 真客户
- 多人协作 · 容易扯皮
- 要反复改 · 要管版本

❌ 别用:
- 5 分钟搞定的小事
- 没人审的练手活
- 只是聊天 · 不出东西

## 六、起手 3 步

### 第 1 步:选 3-7 个神

最小搭配:**悟空 + 白泽 + 你**(干活 + 查 + 拍)

加 1 个:**+ 太白**(干活前先想策略)

完整搭配:7 神(太白 + 悟空 + 白泽 + 唐僧 + 千眼 + 白龙 + 沙僧 + 你)

### 第 2 步:跑 1 个小活试试

用最小搭配,跑 1 个 1 小时内的小活,体验"做事的不审,审的不做事"是什么感觉。

### 第 3 步:跑完写 1 张小卡片

把这次经验记 1 张卡片(模板见 `examples/03-SKILL卡片模板.md`)。20 张攒起来 = 你的办法手册。
