## 贡献方式 / How to Contribute

1. Fork 本仓 · 克隆到本地
2. 创建分支 · `git checkout -b feature/your-feature`
3. 守 3 原则:① 真物驱动 · 反闭门 ② 反 AI 9 通病 ③ 反夸张 / 反阿谀

## PR 守则 / PR Rules

- 反 AI 味 grep("完美 / 必然 / 唯一" = 0)

- 中文 / 中英双语优先
