# SKILL 卡片模板

> 每踩 1 坑 / 发明 1 方法 → 记 1 张 · 20 张攒起来 = 办法手册

---

## 模板(填空版)

| 字段 | 写什么 |
|---|---|
| **序号** | SKILL-XXX |
| **名称** | 4-8 字简称 |
| **立日** | YYYY-MM-DD |
| **立人** | 自己名 · 或哪个神 |
| **什么时候用** | 列 1-3 个具体场景 |
| **怎么用** | 列 1-3 条铁律 |
| **从哪来** | 时锚 + 实战例 + 你的原话(如果是你纠的) |
| **谁能用** | 哪些神 / 哪些项目 |
| **配什么 SKILL** | 跟谁协奏 |
| **防什么通病** | 链回 `02-反AI9通病自检-公开版.md` |
| **重犯前查什么** | 发回复前 grep 什么 |
| **升级路径** | v1.0 → v1.1 → v2.0 |

## 什么时候必记?

✅ 记:
- 踩到 1 个坑(包括小坑)
- 发明 1 个新方法
- 你明示纠了 1 次
- 同类问题第 2 次发生(警告)
- 同类问题第 3 次发生(必记)
- 跨身份大冲突解决后

❌ 不记:
- 一次性琐事
- 已有 SKILL 覆盖
- 没可复用性
