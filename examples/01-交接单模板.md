# 交接单模板

> 跨身份交接必填。消灭"我以为你知道"。

---

## 模板(填空版)

| 字段 | 写什么 |
|---|---|
| **从** | 你的神名 |
| **到** | 下游神名 |
| **时锚** | YYYY-MM-DD HH:MM |
| **内容主题** | 一句话总结 |
| **内容文件** | 真物路径 1 / 路径 2 |
| **版本** | V0.X · 配命名规范 |
| **上游真物** | 需要哪个神的什么才能跑 |
| **外部资源** | API / 服务 / 数据库 / 凭证 |
| **你拍板项** | 哪些需要你先拍 |
| **占位符** | 哪些数字 / 内容待填 |
| **铁律提醒** | 下游神职能边界 / 禁区 / 合规项 |
| **期待回复形式** | 审计报告 / 交付物 / 建议书 |
| **死线** | YYYY-MM-DD HH:MM |

## 反"交接漏字段" 3 通病

| ❌ 通病 | ✅ 正解 |
|---|---|
| "我以为他知道" | 全字段填 · 不留空 |
| "口头交接就行" | 必落文档 · 留痕 |
| "依赖不写" | 显式写 · 避免阻塞 |

---

⭐ **交接单 = 13 字段 + 占位符 + 铁律提醒 = 消灭"我以为" = 跨身份不踩坑**
